# 迁移
```
$ sudo pacman -S nodejs npm
$ git clone -b hexo https://github.com/jiaope/jiaope.github.io
$ cd jiaope.github.io
$ npm install
```
# 使用
```
$ cd jiaope.github.io
$ npx hexo <command>
```
